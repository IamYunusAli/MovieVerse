## Getting Started

**How to clone the project and get started**

To Run the project

```
1. git clone https://github.com/IamYunusAli/MovieVerse.git
2. cd MovieVerse
3. npm i
4. npm run dev
5. use http://localhost:8080/ on your browser
```

To Run test

```
 npm test

```
